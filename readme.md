# RMStream - Plataforma de Streaming

## Descrição
RMStream é uma plataforma de streaming inspirada no YouTube, com interface escura e detalhes em verde. Permite buscar e assistir vídeos do YouTube, explorar canais e descobrir conteúdo relacionado.

## Funcionalidades
- **Busca de vídeos**: Encontre vídeos do YouTube com rolagem infinita
- **Player de vídeo**: Assista vídeos com sugestões relacionadas
- **Perfis de canal**: Visualize informações e vídeos de canais específicos
- **Interface responsiva**: Adaptada para desktop e dispositivos móveis

## Tecnologias
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- API: YouTube Data API v3
- Servidor: XAMPP (Apache)

## Estrutura
- `/views/`: Páginas PHP (index, room, search, channel)
- `/public/css/`: Folhas de estilo
- `/public/js/`: Scripts JavaScript
- `Airules.md`: Documentação e regras do projeto
- `Changelog.md`: Histórico de alterações

## Instalação
1. Instale o XAMPP
2. Clone este repositório na pasta `htdocs` do XAMPP
3. Inicie os serviços Apache no XAMPP
4. Acesse `http://localhost/RMStream/views/index.php` no navegador
